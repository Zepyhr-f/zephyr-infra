- 想了解服务怎么部署时，先看《部署架构与拓扑》。

- 涉及发布流程、灰度与回滚时，查看《CI/CD 与发布策略》。

## 维护规则
- 本目录聚焦上线与交付，不重复维护开发阶段的本地启动说明。
- 环境隔离、配置来源、发布流程变化时，必须同步更新本目录文档。
- 具体发版动作和操作清单应与开发规范中的发版 SOP 保持一致。
